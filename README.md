# 1 Woodland Drive Static Website

Static marketing site for 1WoodlandDrive.com, designed for Cloudflare Pages.

## Local workflow

Copy these files into your local repository:

`/Users/kirkk/Documents/1 Woodland Drive/1woodlanddrive_website`

Then run:

```bash
git add .
git commit -m "Initial property website"
git push -u origin main
```

## Cloudflare Pages settings

- Framework preset: None
- Build command: leave blank
- Build output directory: `/`

## Editing property details

Most shared property information is in:

`assets/js/property.js`

Update contact email, price, status, square footage, and other shared details there.
